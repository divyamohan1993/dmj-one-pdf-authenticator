dmj.one Trust Kit — Quick Guide

Install the dmj.one Root CA once. Then dmj.one‑signed PDFs show as trusted.

Windows — One-click (recommended)

  1. Right-click install-dmj-certificates.bat → Run as administrator
  2. The installer adds:
    • dmj-one-root-ca-r1.(cer/crt) → Trusted Root Certification Authorities
    • dmj-one-issuing-ca-r1.(cer/crt) → Intermediate Certification Authorities
  3. Verify (optional) with certmgr.msc

Manual alternative

Root CA: Double-click dmj-one-root-ca-r1.cer → Install Certificate… → Local Machine (or Current User) → Place all certificates in the following store → Trusted Root Certification Authorities → finish and approve.

Issuing CA (optional): import dmj-one-issuing-ca-r1.cer into Intermediate Certification Authorities.

macOS

  1. Double‑click dmj-one-root-ca-r1.cer (opens Keychain Access)
  2. Select the System keychain (or login)
  3. Drag the certificate in (or File → Import Items…)
  4. Double‑click it → Trust → set Always Trust

Linux (Ubuntu/Debian)

  1. Copy Root (PEM): sudo cp dmj-one-root-ca-r1.crt /usr/local/share/ca-certificates/
  2. Update: sudo update-ca-certificates

Acrobat only (no system changes)

  1. Acrobat → Edit → Preferences → Signatures → Identities & Trusted Certificates → More…
  2. Trusted Certificates → Import → select the Root
  3. Tick Use this certificate as a trusted root and save

Reopen your PDF; it should show as trusted and unchanged.
